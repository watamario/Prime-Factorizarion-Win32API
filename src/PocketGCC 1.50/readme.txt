・このプログラムについて

　入力された自然数を素因数分解したり、指定範囲で素数を探すシンプルなプログラムです。
　しかし、その裏には計算中の"応答なし"回避のためのマルチスレッド化や計算結果の
　エディットボックス表示その他のWindows APIを用いたC++のGUIプログラミングにおける
　基礎的な内容をたくさん詰め込まれています。GUIアプリケーションのテンプレートとしても
　活用できるかと思います。


・ビルド時の要件

　Windows CE端末に、Brain Wiki等を参考にしてPocketGCCをコピーし、付属のbatファイル
　(環境に合わせて特にパス周りを書き換えて下さい)を用いてビルドしてください。


・Windows Embedded CE 6.0のARMv4I? 何それ?

　SHARPのカラー電子辞書、Brainシリーズなどが当てはまります。
　つまり、これは電子辞書で動くプログラムです。言ってみれば、Windows APIによる
　GUIプログラミングを始めたのも、電子辞書用のツールを作りたかったからです。
　電子辞書で動くといっても、新しいモデルでは動かない可能性もあります。
　実行方法や対応機種等の情報はBrain Wikiというサイトを参考にしてください。


・注釈
　ソースコードは、このファイルと同階層にあるwin.cppです。
　実行可能バイナリはAppMain.exeとして生成されます。
