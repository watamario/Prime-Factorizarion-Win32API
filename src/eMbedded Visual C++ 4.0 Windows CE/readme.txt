・このプログラムについて

　入力された自然数を素因数分解したり、指定範囲で素数を探すシンプルなプログラムです。
　しかし、その裏には計算中の"応答なし"回避のためのマルチスレッド化や計算結果のエディットボックス表示その他のWindows APIを用いたC++のGUIプログラミングにおける基礎的な内容をたくさん詰め込んでいます。GUIアプリケーションのテンプレートとしても活用できるかと思います。


・ビルド時の要件

　このプロジェクトは、eMbedded Visual C++ 4.0でのみビルド可能です。
　このIDEは、Windows 2000とXPでしか動かないという代物です。
　また、動作環境はWindows Embedded CE 6.0のARMv4Iを想定しています(一応その他のバイナリも用意していますが､動作確認はとっていません)。


・Windows Embedded CE 6.0のARMv4I？何それ？

　SHARPのカラー電子辞書、Brainシリーズなどが当てはまります。
　つまり、これは電子辞書で動くプログラムです。言ってみれば、Windows APIによるGUIプログラミングを始めたのも、電子辞書用のツールを作りたかったからです。
　電子辞書で動くといっても、新しいモデルでは動かない可能性もあります。実行方法や対応機種等の情報はBrain Wikiというサイトを参考にしてください。


・注釈
　ソースコードは、このファイルと同階層にある1.cppです。
　実行可能バイナリは、デバッグビルドはARMV4IDbgフォルダ、リリースビルドはARMV4IRelフォルダにAppMain.exeとして生成されます。
　なお、その他CPU用のバイナリも「(CPU名)Dbg」、「(CPU名)Rel」の中に「PF_(CPU名).exe」として生成されます｡
